# Apex-OS

Personal operating system for managing training, freelance clients, and productivity.

## Modules

- **Module 1 — Gym Tracker**: Full stack app for logging workouts, exercises, and training sessions.
- **Module 2 — Freelance Operator** _(coming in Phase 4)_: AI-powered client and project manager.

## Stack

- Frontend: Next.js · TypeScript · Tailwind CSS
- Backend: FastAPI · Python · PostgreSQL
- Infra: Docker Compose · JWT Auth
